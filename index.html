Consulta SQL
Faça uma consulta de SQL-Server que exibi a idade em anos a partir da seguinte variável: DECLARE @nasc as varchar(11)='17/04/2008'

CONSULTA :

DECLARE @nasc as varchar(11)='17/04/2008'

SELECT DATEDIFF(YEAR, CONVERT(DATE, @nasc, 103), GETDATE()) AS Idade


EXPLICAÇÃO :

A função DATEDIFF retorna a diferença entre duas datas em um determinado intervalo, neste caso o intervalo é em anos (YEAR). A primeira data é convertida para o tipo DATE usando a função CONVERT, com o formato 103 que representa dd/mm/yyyy. A segunda data é a data atual, obtida pela função GETDATE(). O resultado é exibido na coluna Idade.
